# Rails App

Ruby on Railsチュートリアルを用いて制作した会員登録システム付きのWebアプリケーションです。
いわゆるTwitterのような機能を搭載しています。
(ログイン機能、Remember me機能、マイクロポスト機能、フォロー/リフォロー機能など)

## Description

使い方は簡単。
「Sign up now!」から新規登録するだけ。

制作にあたって学習した内容をQiitaにまとめています。
