# Rails App

Railsで作った会員登録システム付きのアプリケーションです。
ログイン機能、Remember me機能を付けています。

## Description

使い方は簡単。
「Sign up now!」から新規登録するだけ。

使用言語は Ruby/HTML/SCSS です。
